# Template name
Django Security Container Murzagalieva

# Usage
Generates a web interface for URL and content filtering against malicious code injection. 
Enter a URL or HTML content and click «Проверить» to analyze it. 
Results are displayed with a decision (ALLOWED/BLOCKED). 
View statistics on the doughnut chart and check the event log table. 
Edit BLACKLIST_DOMAINS, SUSPICIOUS_PATTERNS, and SHORT_LINK_SERVICES in filter_logic.py to customize filtering rules.
